# Bouképhalos Motorcycle Club

Página web oficial del club de moteros Bouképhalos, inspirado en Alejandro Magno.

## Características

- **10 ubicaciones** en polis griegas antiguas
- **Galería de bares** con imágenes de calidad
- **Mapa interactivo** de Google Maps
- **Noticias** de próximas aperturas
- **Enlace a XCion** para seguir al club en redes sociales

## Despliegue en GitHub Pages

1. Clona este repositorio
2. Haz push a tu rama `main`
3. Ve a Settings → Pages y configura:
   - Source: Deploy from a branch
   - Branch: main / (root)

Tu web estará disponible en: `https://tu-usuario.github.io/boukeephalos-club/`

## Información del Club

Bouképhalos es un club de moteros progresista dedicado a:
- Cuidar de veteranos
- Apoyar a personas en dificultades
- Crear comunidades nómadas en cada polis griega
- Mantener viva la leyenda de Alejandro Magno
